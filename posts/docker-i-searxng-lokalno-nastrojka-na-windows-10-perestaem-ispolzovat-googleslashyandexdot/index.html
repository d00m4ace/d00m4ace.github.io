Установка SearXNG под Windows с помощью Docker:

  1. Установка Docker Desktop:
    • Откройте браузер и перейдите на сайт Docker: https://www.docker.com/
    • Найдите Docker Desktop и нажмите кнопку Download for Windows.
    • Запустите загруженный файл.
    • В окне Configuration, выберите Use WSL 2 instead of Hyper-V (recommended) и Add shortcut to desktop.
    • Нажмите Ok и дождитесь завершения установки.
    • Перезагрузите компьютер.
  2. Установка WSL (Windows Subsystem for Linux):
    • Нажмите Пуск и введите PowerShell.
    • Кликните правой кнопкой мыши по Windows PowerShell и выберите Run as administrator.
    • В окне PowerShell введите команду wsl --install и нажмите Enter.
    • Дождитесь завершения установки и перезагрузите компьютер.

Установка SearXNG как готового образа для Docker

SearXNG можно установить как готовый образ Docker, что упрощает процесс настройки и запуска.

Предварительные условия:

  • Установленный Docker Desktop на вашем компьютере (инструкции по установке Docker Desktop можно найти в предыдущем ответе).

Инструкции:

  1. Загрузите образ SearXNG:

    • Откройте окно PowerShell или CMD.
    • Введите команду docker pull searxng/searxng и нажмите Enter.
    • Дождитесь загрузки образа.

  2. Запустите контейнер SearXNG:

    • Введите команду docker run -d -p 8888:8888 searxng/searxng и нажмите Enter.
      • -d: запустить контейнер в фоновом режиме (detached mode).
      • -p 8888:8888: пробросить порт 8888 контейнера на порт 8888 хоста.
    • Дождитесь запуска контейнера.

  3. Доступ к SearXNG:

    • Откройте браузер и введите адрес http://localhost:8888.
    • Вы увидите главную страницу SearXNG.

Настройки SearXNG:

  • Доступ к настройкам SearXNG осуществляется по адресу http://localhost:8888/preferences.
  • В настройках можно изменить язык поиска, поисковые системы, используемые по умолчанию, и другие параметры.

Дополнительная информация:

  • Вы можете изменить порт, проброшенный на хост, указав другой порт в команде docker run.
  • Дополнительные опции запуска контейнера SearXNG можно найти в документации на GitHub: https://github.com/searxng/searxng-docker

Использование SearXNG:

  • Введите поисковый запрос в строку поиска SearXNG и нажмите Enter.
  • SearXNG выполнит метапоиск, используя выбранные вами поисковые системы.
  • Результаты поиска будут отображены на странице.

Дополнительные возможности:

  • SearXNG поддерживает различные категории поиска, такие как General, Files, Images, IT, Maps, Music, News, Science, Social Media и Videos.
  • Вы можете использовать эти категории для уточнения поискового запроса.

Преимущества SearXNG:

  • SearXNG – это метапоисковая система с открытым исходным кодом.
  • Он не отслеживает ваши поисковые запросы и не показывает рекламу.
  • SearXNG позволяет использовать различные поисковые системы, что дает вам более широкий выбор результатов.
  • SearXNG можно легко установить и настроить с помощью Docker.
