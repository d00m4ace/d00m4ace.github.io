📦 Установка портативных сборок AI Local 3D

В папке находятся:

  • ComfyUI TRELLIS 2
  • Hunyuan3D 2.1
  • CUDA 12.8
  • Visual Studio 2022 Installer

Обе AI-сборки портативные — Python, зависимости и все модели уже внутри.

1️⃣ Сначала установить системные компоненты

🛠️ Установка Visual Studio 2022

Запусти:

vs_Community.exe

После запуска установщика:

  1. Выбери Desktop development with C++ ✅
  2. Справа в Installation details убедись, что отмечены:
  3. ✅ Visual Studio core editor
  4. ✅ Python development
  5. ✅ Desktop development with C++
  6. ✅ Windows 10 SDK (10.0.19041.0)

  7. ✅ MSVC v143 - VS 2022 C++ x64/x86 build tools

  8. Нажми Install

Это необходимо для компиляции CUDA-модулей.

🚀 Установка CUDA 12.8

Запусти файл:

cuda_12.8.0_571.96_windows.exe

Выбери Custom installation и отметь ТОЛЬКО следующие компоненты:

  • ✅ CUDA
  • ✅ Development
  • ✅ Runtime
  • ✅ Visual Studio Integration

❌ НЕ устанавливай: - Nsight Compute - Nsight Systems
- Documentation - Driver components (если уже установлен драйвер)

Нажми NEXT и заверши установку.

2️⃣ Установка ComfyUI TRELLIS 2

Скачать архив

Файлы:

ComfyUI_Trellis2.zip.001
ComfyUI_Trellis2.zip.002

Положить в одну папку.

Распаковать

Используй 7-Zip или WinRAR.

Распаковывать нужно только первый файл:

ComfyUI_Trellis2.zip.001

После распаковки появится папка:

ComfyUI_Trellis2

Запуск

Открой папку и запусти:

run_nvidia_gpu.bat

Интерфейс откроется в браузере:

http://127.0.0.1:8188

⚠️ Если ComfyUI ругается на недостающие ноды:

Если при загрузке воркфлоу появляются ошибки о missing nodes:

  1. Открой ComfyUI Manager (кнопка в меню)
  2. Нажми Install Missing Custom Nodes
  3. Установи все недостающие ноды
  4. Перезапусти ComfyUI

Рабочие воркфлоу

В папке с ComfyUI TRELLIS 2 уже есть готовые рабочие воркфлоу:

  • remove_background.json — удаление фона
  • geometry_texture.json — генерация геометрии + текстуры
  • geometry_only.json — только генерация геометрии
  • workflow_01.png — пример воркфлоу (перетащи в окно ComfyUI)

Как использовать: Перетащи JSON или PNG файл прямо в окно ComfyUI для загрузки воркфлоу.

Все файлы должны лежать в одной папке.

Распаковать

Распаковать только первый файл:

Hunyuan3D2_WinPortable.zip.001

После распаковки появится папка:

Hunyuan3D2_WinPortable

Первый запуск

Открой папку:

Hunyuan3D2_WinPortable

Запусти:

RUN.bat

Вот описание опций RUN.bat (Hunyuan 3D 2 Series Launcher):

⚙️ Доступные настройки:

Select Program (Выбор программы):

  • Hunyuan 3D 2.1 | Better Quality, Slower, No MV — лучшее качество, медленнее, без MV

Enable Texture Generation (Включить генерацию текстур):

  • ✅ Требует: CUDA и MSVC
  • 💾 VRAM требования:
  • Генерация mesh: 3GB VRAM
  • Генерация текстур: 10GB VRAM

Low VRAM Mode (Режим низкой VRAM):

  • ☑️ Включено по умолчанию
  • Не конфликтует с mmgp оптимизацией
  • ⚠️ Отключение значительно увеличит использование VRAM

VRAM Optimization Level (Уровень оптимизации VRAM):

  • Уровень 4 — Low Memory, Low VRAM (выбран по умолчанию)
  • Уровень 5 — максимальная оптимизация (медленнее)
  • Уровень 4 — значительно быстрее, но требует 40GB+ RAM

💾 Кнопка "Save and Start" — сохранить настройки и запустить

Эти настройки позволяют оптимизировать работу под вашу видеокарту и доступную память!

Интерфейс откроется:

http://127.0.0.1:8080

💻 Требования

Минимум:

  • NVIDIA GPU
  • 12 GB VRAM

Рекомендуется:

  • 12–24 GB VRAM
  • 32 GB RAM

✔ Все модели уже загружены — можно сразу начинать работу после установки!
