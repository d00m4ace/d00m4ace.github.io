MP4Tools — это набор бесплатных инструментов для работы с MP4 файлами для Windows, Mac OS, Linux. Включает в себя MP4Joiner для объединения нескольких MP4 файлов без перекодирования и потери качества, и MP4Splitter для разделения одного MP4 файла на несколько частей, также без потери качества. Эти приложения являются открытым программным обеспечением и полностью бесплатны.
