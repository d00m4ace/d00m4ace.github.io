Краткое резюме

1) Общее описание

  • AI Sputnik — Telegram-бот и платформа ИИ как корпоративная онлайн-база знаний.
  • Дает быстрые ответы по данным компании без обучения модели и без встраивания данных; обновления сразу используются.
  • Поддерживает личные и общие базы знаний, доступ для сотрудников.

2) Основные темы

  • Кодовая база (единое хранилище файлов с автоконвертацией в Markdown).
  • Агенты для поиска, пакетной обработки, чатов и интеграций.
  • Макросы для быстрой обработки текстов и кода.
  • Работа с архивами и мультимедиа (распаковка, конвертация, распознавание речи).
  • Управление доступом и историями чатов.
  • Планы расширения и интеграции.

3) Ключевые возможности и инструменты

Кодовая база

  • Автоконвертация загруженных файлов в Markdown.
  • Команды: /files, /search, /download, /delete.
  • Системные агенты:
  • @rag — поиск.
  • @filejob — пакетная обработка.
  • @zip — распаковка и конвертация.

Агент @nethack

  • Работа с распределенной онлайн-базой знаний.
  • Обработка ссылок, режим вопрос/ответ.
  • Управление доступом по группам.
  • Автосинхронизация данных для ИИ.

Агент @chat

  • Диалог с ИИ, загрузка нескольких файлов в общий контекст (до 195 КБ).
  • История до 20 сообщений.
  • Команды управления:
  • /clear_history
  • /export_history
  • /history_info
  • /stop

Макросы @macros

  • Текст:
  • @sum
  • @analyze
  • @translate/_ru
  • @keywords
  • @questions
  • @outline
  • @improve
  • @explain
  • @tasks

  • @entities

  • Код:

  • @code
  • @refactor
  • @comment
  • @tests
  • @security
  • @readme
  • @api
  • @clean

  • @extract

  • Применяются к последнему файлу, можно уточнять запрос, сохраняют результат как новый файл и поддерживают цепочки.

Агент @zip

  • Локальная конвертация PDF, Word, Excel, PowerPoint, HTML в текст.
  • Распаковка .zip и .7z.
  • Единая плоская структура, нормализация имен.
  • Работает без интернета/ИИ.

Агент @rag

  • Контекстный поиск по базе (ориентация на русский язык).
  • Использование:
  • @rag вопрос
  • @rag config — конфигурация.

Агент @filejob

  • Пакетная ИИ-обработка по селекторам файлов (маски, диапазоны, списки).

Аудио

  • До 20 МБ; авто-распознавание речи в Markdown.
  • Рекомендации по сжатию (Audacity: Ogg Vorbis, Mono, 22050/44100 Hz, Quality 0–2).
  • Поддержка транскрипции и суммаризации длительных записей.

4) Ключевые преимущества

  • Быстрый доступ к знаниям компании прямо в Telegram.
  • Отсутствие обучения модели и встраивания данных; актуальные данные доступны мгновенно.
  • Приватность: локальная конвертация документов, без интернета для @zip.
  • Управление доступом к документам на уровне групп.
  • Масштабируемость через агентов и макросы; пакетная обработка и поиск.
  • Удобные инструменты для работы с файлами, кодом и аудио.

5) Планы развития

  • Новые агенты по запросам компаний.
  • HR-интеграция с Huntflow для обработки баз резюме (сортировка, суммаризация, поиск, отбор).
  • Новые провайдеры знаний для @nethack.
  • Специализированные агенты для мозговых штурмов с ролями.
  • Интеграция с teamwork.com для постановки задач.

6) Выводы

  • AI Sputnik — централизованный ИИ-инструмент для корпоративной базы знаний и обработки контента, ориентированный на русский язык.
  • Покрывает сценарии поиска, Q&A, пакетной и диалоговой работы с файлами/кодом/аудио.
  • Обеспечивает контроль доступа и оперативную актуализацию данных, активно развивается через новые агенты и интеграции.
